# homebrew-reddio

Homebrew tap for `reddio`, a command-line interface for Reddit written in POSIX sh.

## Install

```sh
brew tap zero-click/homebrew-reddio
brew install reddio
```

## Versioned installs

```sh
brew install reddio@0.52
brew install reddio@0.46
```

Homebrew can keep multiple versions installed, but only one linked on your `PATH` at a time. Switch with:

```sh
brew unlink reddio reddio@0.46 reddio@0.52
brew link --force --overwrite reddio@0.46
```

## Formulae

- `reddio`
- `reddio@0.52`
- `reddio@0.46`
